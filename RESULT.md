# 실험 결과 정리 (RESULT)

본 문서는

`train_all.py` (전체 발전소 기준)와

`train_by_plant.py` (발전소별 기준)

두 가지 실험 결과를 정리한 문서이다.

---

## 1. 전체 발전소 기준 결과

### 1-1. 실험 설정 요약

- 대상: 모든 태양광 발전소 데이터 통합
- 시간 범위: 주간 시간대 (06~18시)
- 모델: XGBoost Regressor
- 입력 변수:
    - 설비용량(MW)
    - 월
    - 시간
    - 요일
- 타깃 변수: 시간당 발전량 (kWh)

---

### 1-2. 성능 지표

| 지표 | 값 |
| --- | --- |
| MAE | 470.16 kWh |
| RMSE | 938.07 kWh |
| R² | 약 0.77 |

> 전체 발전소 데이터를 하나의 모델로 학습했을 때
> 
> 
> 태양광 발전의 **시간대별 공통 패턴**을 비교적 잘 학습함을 확인하였다.
> 

---

### 1-3. 전체 발전소 예측 결과 시각화

### ▶ 실제 발전량 vs 예측 발전량 산점도

![](/images/train_all_pred.png)
- 대각선(y = x) 부근에 점들이 분포
- 시간 정보와 설비용량을 기반으로 한 **베이스라인 모델로서는 양호한 성능**
- 발전소 규모 차이가 모델 성능에 큰 영향을 미침

---

### 1-4. 시간대별 평균 발전량 패턴
![](/images/train_all_time.png)
- 오전 9시 이후 발전량 증가
- 정오 전후 최대 발전량 형성
- 태양광 발전의 전형적인 일중 패턴 확인

---

### 1-5. 해석

- 전체 모델의 비교적 높은 R²는
    
    **발전소 규모 차이와 시간대 효과를 동시에 학습한 결과**
    
- 단일 모델로는 발전소별 세부 변동성까지 설명하기에는 한계 존재

---

- 대상: 발전소별 개별 데이터

### 2-4. 발전소별 모델 분석

- 발전소별로 발전 패턴과 예측 성능 차이가 큼
- 일부 발전소는 발전량 변동성이 작아 R²가 매우 낮게 나타남
- 이는 **모델 실패가 아닌 데이터 특성**으로 판단됨

---

## 3. 전체 모델 vs 발전소별 모델 비교

| 항목 | 전체 발전소 모델 | 발전소별 모델 |
| --- | --- | --- |
| 데이터 수 | 많음 | 상대적으로 적음 |
| R² | 높음 (≈0.7) | 발전소별 편차 큼 |
| 해석 | 공통 패턴 파악 | 개별 특성 분석 |
| 실무 활용 | 베이스라인 | 운영 단위 분석 |

---

## 4. 종합 결론

- 태양광 발전량은 **시간대 정보에 크게 의존**
- 전체 모델은 베이스라인으로 유효하지만,
    
    발전소별 세부 예측에는 한계 존재
    
- 발전소 단위 분석을 통해
    
    **기상 데이터 및 추가 변수의 필요성**을 확인
    

---

## 5. 향후 개선 방향

- 기상 데이터(일사량, 구름량) 결합
- 발전량 대신 **발전률(kWh/MW)** 기준 예측
- 시계열 모델(LSTM 등) 적용
- 발전소별 이상치/설비 이상 탐지
