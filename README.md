# 신재생에너지 태양광 발전량 예측 프로젝트

**(한국서부발전 태양광 발전 데이터 기반)**

## 1. 프로젝트 개요

본 프로젝트는 한국서부발전에서 제공한 **2022년 태양광 발전소별 시간 단위 발전량 데이터**를 활용하여

태양광 발전의 **시간적 패턴을 분석하고 발전량을 예측**하는 것을 목표로 한다.

특히,

- 전체 발전소를 하나의 모델로 학습한 경우
- 발전소별로 모델을 분리하여 학습한 경우

를 비교함으로써 **단일 모델의 한계와 발전소별 특성 차이**를 확인하고자 했다.

본 프로젝트는 **하드웨어(IoT 디바이스) 없이 데이터 분석 및 머신러닝 중심**으로 구성된 개인 프로젝트이다.

---

## 2. 사용 데이터

- **데이터 출처**:
    
    한국서부발전(주)_(AI친화)신재생 에너지 발전량 데이터 (2022년)
    
- **데이터 특징**
    - 태양광 발전소별 시간 단위 발전량 데이터
    - 주요 컬럼:
        - 발전기명
        - 연월일
        - 시분초
        - 발전량
        - 설비용량(MW)
        - 주소
- **전처리 내용**
    - `24:00:00` 비정상 시간 제거
    - 날짜 + 시간 결합하여 `datetime` 생성
    - 월 / 시간 / 요일 파생 변수 생성
    - 태양광 특성을 고려하여 **주간 시간대(06~18시)만 사용**
    - 발전량 단위 보정 (Wh → kWh)

---

## 3. 프로젝트 구성

본 프로젝트는 분석 목적에 따라 **두 가지 모델링 방식**으로 구성된다.

### ① 전체 발전소 기준 모델 (`train_all.py`)

- 모든 발전소 데이터를 하나로 합쳐 학습
- 태양광 발전의 **공통적인 시간대 패턴** 파악
- 베이스라인 성능 확인

### ② 발전소별 기준 모델 (`train_by_plant.py`)

- 발전소 단위로 데이터를 분리하여 개별 학습
- 발전소별 발전 패턴 및 예측 성능 비교
- 단일 모델의 한계 분석

## 4. 사용 기술 스택

- **언어**: Python
- **라이브러리**
    - pandas, numpy
    - matplotlib
    - scikit-learn

    - Gradient Boosting 기반 회귀 모델 (XGBoost)

---

## 5. 모델 학습 방식

- 입력 변수 (Features)
    - 설비용량 (MW)
    - 월
    - 시간
    - 요일
- 타깃 변수 (Target)
    - 시간당 발전량 (kWh)
- 데이터 분할
    - Train / Test = 80% / 20%
- 평가 지표
    - MAE (평균 절대 오차)
    - RMSE (평균 제곱근 오차)
    - R² (결정계수)

---

## 6. 주요 결과 및 분석

### ✔ 전체 발전소 기준

- R² 약 **0.7 수준**
- 시간대에 따른 발전량 패턴이 뚜렷하게 나타남
- 발전소 규모 차이가 모델 성능에 큰 영향을 미침

→ **베이스라인 모델로서 유의미한 결과**

---

### ✔ 발전소별 기준

- 발전소별로 R² 편차가 큼
- 일부 발전소는 예측이 비교적 잘 되지만,
    
    일부 발전소는 변동성이 커 성능이 낮음
    

→ 발전량은 **시간 정보만으로는 한계가 있으며**,

기상 정보(일사량, 구름량 등)의 필요성을 확인

---

## 7. 시각화 결과

- 실제 발전량 vs 예측 발전량 산점도
- 발전소별 **시간대 평균 발전량 그래프**
    - 태양광 발전 특성상 정오 전후 최대 발전량 확인
    - 발전소별 패턴 차이 시각적으로 확인 가능

---

## 8. 한계점 및 개선 방향

- 기상 데이터(일사량, 기온, 구름량) 미포함
- 발전소별 설비 상태, 음영, 유지보수 정보 부재
- 단일 연도 데이터 사용

**향후 개선 방향**

- 기상 데이터 결합
- 발전률(kWh/MW) 기준 예측
- 시계열 모델(LSTM 등) 적용

---

## 9. 프로젝트 의의

- 신재생에너지 태양광 발전 데이터를 활용한 **실제 운영 관점 분석**
- 단일 모델과 발전소별 모델 비교를 통한 **문제 구조 이해**
- 단순 예측이 아닌 **모델의 한계와 데이터 특성 분석**에 초점
