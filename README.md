# 태양광 발전량 예측 및 시각화 프로젝트

## 1. 프로젝트 개요
본 프로젝트는 2022년도 태양광 발전소의 시간별 발전량 데이터를 이용하여  
시간 기반 특성을 학습하고 발전량을 예측한 뒤 결과를 시각화하는 것을 목표로 한다.

기상 데이터가 포함되지 않은 환경을 가정하여  
시간 정보만으로 발전량 패턴을 분석하였다.

---

## 2. 사용 데이터
- 데이터 출처: 한국서부발전(주) 신재생 에너지 발전량 데이터
- 주요 컬럼
  - 발전기명
  - 연월일
  - 시분초
  - 발전량
  - 용량_메가

---

## 3. 데이터 전처리
- 연월일과 시분초를 결합하여 datetime 컬럼 생성
- 시간 기반 파생 변수 생성
  - 월(month)
  - 시간(hour)
  - 요일(dayofweek)
- 학습에 필요한 컬럼만 선택하여 데이터 구성

---

## 4. 학습 모델
- RandomForestRegressor
- 입력 변수
  - 용량_메가
  - month
  - hour
  - dayofweek
- 타겟 변수
  - 발전량

---

## 5. 학습 결과
- MAE(Mean Absolute Error)를 이용한 오차 평가
- 실제 발전량과 예측 발전량 비교 시각화
- 시간대별 평균 발전량 시각화

---

## 6. 결과 시각화
- 실제 발전량 vs 예측 발전량 산점도
- 시간대별 평균 발전량 변화 그래프

---

## 7. 한계 및 개선점
- 본 데이터셋에는 기상 정보가 포함되어 있지 않아
  시간 기반 특성만을 활용한 예측을 수행하였다.
- 향후 기상 데이터와 결합할 경우 예측 성능 향상이 기대된다.
